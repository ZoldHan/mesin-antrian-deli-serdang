## 🎯 QUICK START GUIDE

### **Untuk Peserta Latsar (Sidang):**

1. **Baca:** `DOKUMENTASI/README-UTAMA.md` ← Start here!
2. **Pelajari:** `SIDANG/presentasi-sidang.md`
3. **Cetak:** `SIDANG/laporan-aktualisasi.md` (convert ke PDF)
4. **Demo:** Jalankan server → `npm start`
5. **Siapkan:** Checklist di `MATERI-SIDANG-CHECKLIST.md`

### **Untuk IT Staff (Implementasi):**

1. **Baca:** `DOKUMENTASI/PANDUAN-INSTALASI.md`
2. **Install:** Follow step-by-step
3. **Test:** `DOKUMENTASI/TROUBLESHOOTING.md`
4. **Deploy:** Production server
5. **Train:** Staff pakai `DOKUMENTASI/PANDUAN-PENGGUNAAN.md`

### **Untuk Masyarakat (Pengguna):**

1. **Baca:** `DOKUMENTASI/PANDUAN-PENGGUNAAN.md` (Bagian 1)
2. **Akses:** http://localhost:3000
3. **Daftar:** Isi form → Dapat nomor antrian
4. **Tunggu:** SMS notifikasi
5. **Datang:** Sesuai jadwal

### **Untuk Staff Loket:**

1. **Baca:** `DOKUMENTASI/PANDUAN-PENGGUNAAN.md` (Bagian 2)
2. **Login:** http://localhost:3000/admin.html
3. **Panggil:** Antrian per loket
4. **Update:** Status selesai
5. **Laporan:** Export Excel harian
